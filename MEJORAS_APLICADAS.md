# Mejoras aplicadas al diseño UNIVBRAKES

Se aplicaron los comentarios recibidos sobre diseño y maquetación:

- Se mantuvo un concepto gráfico más parecido al de redes sociales: fondo gris, rojo, amarillo, negro y blanco.
- Se redujo el uso de fondos y efectos distintos para que el sitio se vea más uniforme.
- Se dejaron solo dos estilos tipográficos principales: uno fuerte para títulos y botones, otro limpio para textos.
- Se eliminó la barra lateral de redes sociales para evitar repetición.
- Se quitaron las redes del inicio; ahora se concentran al final de la página.
- Se quitó el botón flotante de WhatsApp para evitar que tape contenido o se repita visualmente.
- Se enderezó la imagen principal y se eliminó el cuadro sobrepuesto que se veía chueco.
- Los datos rápidos ya no parecen botones; ahora se muestran como una franja informativa.
- Las tarjetas de sucursales ya no repiten redes sociales; solo muestran información útil de atención.

Para actualizar la página en Netlify o hosting, sube nuevamente toda la carpeta `univbrakes_web`.
