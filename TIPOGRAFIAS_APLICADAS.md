# Tipografías aplicadas

Se ajustó la página para usar dos familias tipográficas principales:

- Títulos, frases grandes y encabezados: estilo condensado tipo Bebas Neue / Intro Demo.
- Textos, formularios, menú y descripciones: Montserrat para mejor lectura.

Nota técnica: no se incluyen archivos de fuente dentro del ZIP. La página usa Bebas Neue desde Google Fonts y puede usar Intro Demo si está instalada localmente en el equipo del visitante. Si más adelante cuentas con licencia web para publicar esas fuentes, se pueden conectar desde el hosting.
